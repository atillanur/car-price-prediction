# Used Car Price Prediction

This project aims to estimate the price of used cars using machine learning techniques.  
The model predicts car prices based on various vehicle features such as engine size, horsepower, fuel type, and more.

---

## Project Overview

- Data was cleaned and processed using **pandas**
- Categorical features were encoded using **OneHotEncoder**
- A **Random Forest Regressor** was used for prediction
- Model performance was evaluated using standard regression metrics

---

## Model Performance

- **R² Score:** 0.958  
- **Mean Absolute Error (MAE):** ~1250  

The results show that the model predicts prices with high accuracy and maintains a strong correlation between real and predicted values.
